    <h1 id=“label-Ruby+client+library+for+FreeIPA+JSON+API”>Ruby client library for FreeIPA JSON API<span><a href=“#label-Ruby+client+library+for+FreeIPA+JSON+API”>&para;</a> <a href=“#top”>&uarr;</a></span></h1>

    <p>This is a simple Ruby client library that allows to interact with the FreeIPA JSON API. It currently only supports authenticating via Kerberos/GSSAPI tickets.</p>

    <p>Pull requests to add additional API features are very welcome. I only implemented what I needed.</p>

    <h2 id=“label-Install”>Install<span><a href=“#label-Install”>&para;</a> <a href=“#top”>&uarr;</a></span></h2>

    <p>To install it simply issue the following command:</p>

    <pre class=“ruby”><span class=“ruby-identifier”>gem</span> <span class=“ruby-identifier”>install</span> <span class=“ruby-identifier”>ipa</span><span class=“ruby-operator”>-</span><span class=“ruby-identifier”>ruby</span> </pre>

    <h2 id=“label-Usage”>Usage<span><a href=“#label-Usage”>&para;</a> <a href=“#top”>&uarr;</a></span></h2>

    <p>You can optionally pass a ca_cert keyword argument specifying the path to the FreeIPA CA certificate. Default is /etc/ipa/ca.crt. require &#39;ipa/client&#39; ipa = IPA::Client.new(host: &#39;ipa.example.org&#39;) </p>

    <p>Note that additional parameters can be passed via the params keyword argument.</p>

    <p>Add a host (with a random password): ipa.host_add(hostname: &#39;foo.example.org&#39;, force: true, random: true, all: true) </p>

    <p>Add a host (with a specific password) ipa.host_add(hostname: &#39;foo.example.org&#39;, force: true, userpassword: &#39;bar&#39;, all: true) </p>

    <p>Delete a host: ipa.host_del(hostname: &#39;foo.example.org&#39;) </p>

    <p>Show a host: ipa.host_show(hostname: &#39;foo.example.org&#39;, all: true) </p>

    <p>Find hosts: ipa.host_find(all: true, params: {:in_hostgroup =&gt; true}) </p>

    <p>Check if a host exists if ipa.host_exists?(&#39;foo.example.org) puts &quot;Yep :)&quot; else puts &quot;Nope :(&quot; end </p>
